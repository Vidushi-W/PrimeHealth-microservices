# Patient Service

This is the Patient Microservice for the PrimeHealth application.
It handles all patient-related data and requests using a modular architecture.

## Structure Overview

- `src/config/`: Database connections
- `src/controllers/`: Route handlers and logic
- `src/models/`: Database schemas
- `src/routes/`: API endpoints definitions
- `src/middleware/`: Authentication and validation middleware
- `src/app.js`: Main Express application setup

## Getting Started

1. Set up your `.env` variables.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the service in development mode.
